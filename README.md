# Workshop-Springboot4-jpa

## 📋 Sobre o Projeto

Este é um projeto de estudo desenvolvido durante um workshop de **Spring Boot**, focado no aprendizado de:
- JPA (Java Persistence API)
- Banco de dados H2
- Camada de serviço e injeção de dependência
- Repositórios Spring Data JPA
- Tratamento de exceções

## 🚀 Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- Banco de dados H2 (ambiente de teste)
- Maven

## 📁 Estrutura do Projeto

- **Entidades**: Usuario, Turma
- **Repositórios**: Interface com Spring Data JPA
- **Serviços**: Camada de serviço com registro de componentes
- **Tratamento de exceções**: findById, update, etc.

## 🔧 Funcionalidades Implementadas

- ✅ CRUD de Usuario (inserção, deleção, edição PUT)
- ✅ Tratamento de exceções para update e findById
- ✅ Enum PerfilDoUsuario
- ✅ Entidade Turma com método AdicionaUsuarios

## 🗄️ Banco de Dados

O projeto utiliza o **H2 Database** em modo de teste, permitindo fácil execução sem configuração adicional.
